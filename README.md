# Superstore Sales Dataset Analysis

## Overview

This project analyzes the **Sample - Superstore** dataset using Python and Pandas. The objective is to explore the dataset, understand its structure, and answer basic data analysis questions.

## Questions Solved

1. How many rows and columns does the dataset have?
2. What are the column names in the dataset?
3. Are there any missing values?
4. What is the highest sales value?
5. Which product category appears the most number of times?
6. What does the dataset seem to be about?

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Dataset

**Dataset Name:** Sample - Superstore.csv

## Key Findings

* Total Rows: **9994**
* Total Columns: **21**
* Missing Values: **None**
* Highest Sales Value: **22638.48**
* Most Frequent Category: **Office Supplies**

## Repository Contents

* `Superstore_Dataset_Analysis.ipynb` – Jupyter Notebook containing the complete analysis.
* `Sample - Superstore.csv` – Dataset used for the analysis.
* `README.md` – Project documentation.

## Project Purpose

This project was completed as part of a Data Analytics learning task. The analysis focuses on understanding the dataset structure, identifying key insights, and practicing fundamental data exploration techniques using Python and Pandas.

## Conclusion

The dataset contains sales records of a retail superstore, including customer information, product details, sales transactions, discounts, and profit data. It can be used to study sales trends, customer behavior, product performance, and overall business insights.
